# 2023 하반기 채용공고 프론트엔드 개발자 Toy-Project

## 요구 사항

1. 로그인 후 새 화면에 Swagger > InformationResponse에 있는 이미지  (세션 유지한 경우에만 출력)
2. GNB Component, 클릭 시 하이라이트, 클릭되지 않으면 예시와 같이 어둡게
3. 본인의 github 또는 Notion 주소 연결. 없는 경우 자사 홈페이지 www.buttercorp.kr 연결
4. 3s 간격으로 배너 이미지 롤링
5. 회원가입 완료 후 자동 창 닫기, 전체 필수 값 API 연동 작업
6. 팝업 창 닫기, Login 팝업창 닫고 Signup 팝업창 출력, 클릭 시 Login 팝업창 메인 가운데 출력
